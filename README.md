\# Academic Project: Patient Classification Analysis

\*\*Institution:\*\* NOVA IMS Lisboa | \*\*Course:\*\* Data Mining 2



\## 📋 Summary

A classification project in a large healthcare dataset of 75000 rows and over 30 columns. The features of the given dataset proved to be weakly correlated to the target variable, leading to poor scores especially in recall and F1 scores, and a tendency to exhibit a strong bias towards one class. Model complexity also did not translate into improved performance, and even with hyperparameter tuning and advanced techniques such as stacking and neural networks, results were insufficient. This was an excellent example on how to deal with a dataset that lacks predictive potential and how to still extract as much as possible from it, using all available corrective actions.
